git客户端密码位置要用token登录才可以

生成自己的token

  1. 在个人设置页面，找到Setting

  2. 选择开发者设置Developer setting

  3. 选择个人访问令牌Personal access tokens，然后选中生成令牌Generate new token

  4. 设置token的有效期，访问权限等

  5. 设置token的有效期，访问权限等

  6. 在git客户端输入 git config –system –unset credential.helper即可重新输入密码
